NAME

joint-indels -- Usage: joint-indels [OPTIONS]

SYNOPSIS

joint-indels alignments file trees file [OPTIONS]

DESCRIPTION

Usage: joint-indels [OPTIONS]

ALLOWED OPTIONS:

--help
produces help message
--subsample arg (=10)
factor by which to sub-sample trees
--partition arg
find indels along internal branch that bi-partitions given taxa (::...)
--alphabet arg
set to 'Codons' to prefer codon alphabets
--verbose
Output more log messages on stderr.
