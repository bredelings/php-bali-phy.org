NAME

joint-parsimony - Usage: joint-parsimony [OPTIONS]

SYNOPSIS

joint-parsimony alignments file trees file [OPTIONS]

DESCRIPTION

Usage: joint-parsimony [OPTIONS]

ALL OPTIONS:

--help
produces help message
--subsample arg (=10)
factor by which to sub-sample trees
--alphabet arg
set to 'Codons' to prefer codon alphabets
